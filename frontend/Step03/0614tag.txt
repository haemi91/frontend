파일명 정할 때 주의 사항
  1. 공백 X
  2. 한글 X
  3. 특수 문자 _
  4. 첫글자는 알파벳 - html 파일 만들 경우 

페이지 경로
  1. 상대경로 : 현재 웹페이지가 있는 폴더를 기준으로 원하는 파일까지 접근하는 경로
  2. 절대경로 : root(맨 처음 폴더)에서 부터 접근하는 경로

img
  jpg png gif bmp 
  svg
  <img src="이미지파일경로" alt="대체텍스트">
  대체텍스트 : 해당 이미지파일이 잘렸을때 대체해서 나타낼 문자열
              웹 접근성을 위해 시각장애인용 스크린리더에 나타낼 내용
video
  mp4 avi mov mkv wmv mpeg ts tp ogv
audio
  mp3 ogg wav flac aac mqa m4a wma

iframe : 현재 웹페이지에서 다른 웹페이지를 출력할때 사용하는 태그
