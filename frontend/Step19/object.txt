객체(Object)
    - 대상 , 일상생활에서 볼 수 있는 것들을 코드로 표현
    - 변수와 값, 함수를 가지고 있음
    - 객체를 이용하면 이미 만들어진 데이터와 함수를 사용할 수 있음

객체 사용 방법
1. 객체를 생성(new 객체명(), 객체명.)
2. 객체.변수 = 데이터 <--- 생성된 객체에 데이터를 저장
   객체.setDate(저장할 데이터) <--- 함수를 이용해서 데이터를 저장
3. 객체.함수명(필요한 데이터를 작성)

내장 객체 --> 자바스크립트에서 지원하는 기능

날짜, 수학, 윈도우, 문자열, 배열, 사용자 정보
