목록 태그 스타일 변경
배경
  background-color : 색상값;
  background-origin: 배경이미지 채울 영역;
  background-clip : 배경색 채울 영역;
      border-box : 테두리 영역까지 채움 - 기본값
      padding-box : padding 영역까지 채움
      content-box : 내용 영역까지 채움

  background-size : 배경 크기값
        cover : 가로세로 비율 중 작은거 기준으로 채움, 
                큰 비율에 해당하는 이미지가 짤림(가로나 세로영역 일부가 잘림)
                이미지가 영역 전체를 채움
        contain : 가로세로 비율 중 큰거 기준으로 채움, 
                  이미지가 영역 전체를 채우지는 못함
   
box-modeling
  margin : 테두리 바깥쪽 여백
  padding : 테두리 안쪽 여백
  border : 테두리
  content : 실제 내용이 표시되는 영역

border : 두께 선종류 색상
border-radius: 170px; 테두리 둥글게

position 
    absolute : html 문서를 기준으로 고정
    fixed : 웹 브라우저 기준으로 고정
    relative : 본래 출력을 해야될 위치에 기준
    static : 본래 배치 순서에 맞게 출력

z-index : 배치되는 우선 순위 값이 높으면 맨앞에 출력
